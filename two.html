Q1. What does the command pwd do?

Print Working Directory - displays the current path.
1 Mark

Q2. Which command is used to list files and directories, including hidden ones?

ls -a
1 Mark

Q3. What is the difference between absolute and relative paths?

Absolute starts from root (/); Relative starts from current location.
1 Mark

Q4. How do you change the permissions of a file to give the owner read, write, and execute permissions, while giving no permissions to group and others?

chmod 700 filename
1 Mark

Q5. What command would you use to view the last 20 lines of a file called log.txt?

tail -n 20 log.txt
1 Mark

Q6. On RHEL, which command is used to manage services start, stop, enable, or check the status of a service?

systemctl
1 Mark

Q7. On RHEL, what command installs the Nginx web server?

sudo dnf install nginx OR sudo yum install nginx
1 Mark

Q8. What is the difference between a hard link and a soft (symbolic) link?

Hard links point to the inode; Soft links point to the filename.
